# 조건문
**조건문**: 주어진 조건에 따라 명려의 실행 여부를 결정하는 문장.  
&nbsp;&nbsp;&nbsp;**선언**: if(조건){명령}
> 괄호안에 조건이 만족한다면 중괄호 안에 명령이 실행된다.
> 명령문이 한 줄인 경우에는 중괄호는 생략이 가능하다.
<pre>#include <stdio.h>
int main()
{
	int i;

	printf("정수를 입력하세요.");
	scanf_s("%d", &i);
	if (i == 7)
		printf("7을 입력하였습니다.");
	if (i == 8)
	{
		printf("8을 입력하였습니다.");
		printf("8을 입력하였군요");
	}
}
</pre>

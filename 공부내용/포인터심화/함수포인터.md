# 함수 포인터 
**함수 포인터**: 함수를 가리키는 포인터  
&nbsp;&nbsp;&nbsp;**선언**: 리턴타입 *변수명(인수목록)  
> 함수 포인터와 구별하기 위해 일반 포인터를 데이터 포인터라 부르기도 한다.  
> 인수의 타입만 같으면 돼고 인수명은 똑같을 필요 없다.  
> 단독으로 사용된 함수명은 함수의 시작 번지를 나타내는 포인터 상수이다.  
> 원형만 일치한다면 함수 포인터가 여러개의 함수를 가리킬 수 있다.

<pre>#include <stdio.h>
int func(int a)
{
     return a*2;
}

void main()
{
     int i;
     int (*pf)(int a);
     pf=func;

     i=(*pf)(2);
     printf("%d\n",i);
}</pre>

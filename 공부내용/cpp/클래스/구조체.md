# 멤버 함수
**멤버 함수**: 구조체에 포함된 함수  
> 구조체에 포함된 변수는 멤버 변수라고 부른다.  
> 다른 언어에서는 멤버 변수를 필드(Field), 멤버 함수를 메소드(Method)라고 부르기도 하는데 같은 뜻이다.  

<pre>
#include < Turboc.h >

struct Position
{
     int x;
     int y;
     char ch;
     void OutPosition() {

          gotoxy(x, y);
          putch(ch);
     }
};

void main()
{
     Position Here;
  
     Here.x=30;
     Here.y=10;
     Here.ch='A';
     Here.OutPosition();
}
</pre>

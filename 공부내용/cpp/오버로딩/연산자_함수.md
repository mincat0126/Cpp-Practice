# 멤버 연산자 함수
&nbsp;&nbsp;&nbsp;**선언**: 리턴타입 Class::operator 연산자(인수 목록){내용;}  
> 연산자 자리에는 +, -, *, /, <<, != 등 대부분의 연산자 기호가 올 수 있다.  
> 연산자 함수의 인수란 피연산자를 의미한다.  

**<연산자 오버로딩 규칙>**
1. 원래 C++ 언어가 제공하는 기존 연산자만 오버로딩의 대상이며 C++이 제공하지 않는 연산자를 임의로 만들 수는 없다.
2. 이미 존재하는 연산자 중에도 오버로딩의 대상이 아닌 것들이 있다.(. :: ?: .* sizeof new delete 등)
3. 기존 연산자의 기능을 바꾸더라도 연산자의 피연산자의 개수와 우선 순위는 변경할 수 없다.
4. 한 클래스가 하나의 연산자를 여러 가지 피연산자 타입에 대해 오버로딩할 수 있다.
5. 오버로딩된 연산자의 피연산자 중 적어도 하나는 사용자 정의형이어야 한다.

<pre>
#include < iostream >
  
class Complex
{
private:
     double real;
     double image;

public:
     Complex() { }
     Complex(double r, double i) : real(r), image(i) { }
     void OutComplex() const { printf("%.2f+%.2fi\n",real,image); }
    const Complex operator +(const Complex &T) const {
        Complex R;
        R.image = image + T.image;
        R.real = real + T.real;
        return R;
    }
};

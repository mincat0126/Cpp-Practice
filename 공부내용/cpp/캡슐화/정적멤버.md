# this
**this**: 호출한 객체의 번지를 가리키는 포인터 상수  
> 일반적으로 Class형의 멤버 함수들은 Class * const this를 받아들이며 this로부터 객체의 고유한 멤버를 액세스할 수 있다.    
> 멤버 함수 호출시에 this를 암시적으로 전달하는 호출 규약을 thiscall이라고 하는데 모든 멤버 함수에 자동으로 적용된다.  
> 가변 인수를 취하는 멤버 함수는 cdecl 호출 규약을 사용한다.  
> this에 대한 모든 관리는 컴파일러가 알아서 처리한다.  
> 자기 자신에 대한 레퍼런스를 리턴할 때는 *this 표현식을 사용한다.  
> 객체가 자신을 스스로 삭제하고자 할 때도 this 키워드를 쓴다.  
> 동적으로 할당된 객체일경우 치명적인 에러나 프로그램 종료시 스스로 자살하고자 할 때 delete this; 한 줄이면 가볍게 생을 마감할 수 있다.

# 정적 멤버
**정적 멤버 변수**: 클래스의 바깥에 선언되어 있지만 클래스에 속하며 객체별로 할당되지 않고 모든 객체가 공유하는 멤버이다.
&nbsp;&nbsp;&nbsp;**선언**: static 타입 변수명;
> 개별 객체와는 직접적인 상관이 없고 객체 전체를 대표하는 클래스와 관련된 정보를 저장하는 특수한 멤버이다.

<pre>
#include < iostream >

class Count
{
private:
    int Value;
	static int Num;

public:
    Count() { Num++; }
    ~Count() { Num--; }

    void OutNum()
    {
        printf("현재 객체 개수 = %d\n", Num);
    }
};
int Count::Num = 0;

void main()
{
    Count C, *pC;                           //<실행결과>
	C.OutNum();                             // 현재 객체 개수 = 1
	pC = new Count;                         
	pC->OutNum();                           // 현재 객체 개수 = 2
	delete pC;                              
	C.OutNum();                             // 현재 객체 개수 = 1
	printf("크기 = %d\n", sizeof(C));       // 크기 = 4
}
</pre>

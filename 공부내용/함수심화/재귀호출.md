# 재귀 호출
**재귀 호출**: 자기 자신을 호출하는 형식  
> 각 호출 인스턴스의 모든 정보는 함수의 지역변수에 완벽하게 보존된다.

<pre>#include < stdio.h >

int Factorial(int n)
{
     if (n <= 1) 
     {
          return 1;
     } 
     else 
     {
          return n*Factorial(n-1);
     }
}

void main()
{
     printf("1~5까지의 곱=%d\n",Factorial(5));
}</pre><br><br><br>
